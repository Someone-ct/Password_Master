# Password_Master
**Password Master** is a powerful and user-friendly tool designed to help users create, analyze, and verify the security of their passwords. It provides essential features that improve password strength and help users understand whether their passwords are truly secure.

The program includes three main features:

🔐 **Password Generator**
Password Master can generate strong and secure passwords automatically. Users can also customize the generation process by selecting the password length and choosing whether to include uppercase letters, lowercase letters, numbers, and special characters. This ensures the creation of complex passwords that are difficult to guess or crack.

🛡 **Password Strength Checker**
This feature allows users to evaluate the strength of any password. The program analyzes the password and provides a clear assessment of its security level, helping users understand whether their password is weak, moderate, or strong.

🔎 **Wordlist Password Search**
The most important feature of Password Master is the ability to search for a password within a wordlist. This allows users to check whether their password appears in common password lists often used in security testing and password-cracking attempts. If the password is found in such lists, it indicates that the password may not be secure and should be changed.

✨ With these features, **Password Master** helps users build stronger passwords and improve the overall security of their online accounts.
